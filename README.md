# notambourine/homebrew-tap

Homebrew tap for my tools. Install with:

```sh
brew tap notambourine/tap
brew install <tool>
```

| Tool | Install | Source |
|---|---|---|
| [mdql](https://github.com/notambourine/mdql) | `brew install mdql` | Schema-driven CRUD + FTS + wiki graph over markdown notes |

Formulas under `Formula/` are auto-generated and committed by
[GoReleaser](https://goreleaser.com) from each tool's release pipeline.
Don't hand-edit — changes get overwritten on the next release.
